About Me

I began my journey into programming in college taking programming classes in C++ and Python. You might think that I studied programming and that I got a CS degree, but I didn’t! I always liked coding and it was a great hobby but I actually studied Mechanical Engineering, and I don’t regret it! I enjoyed all the things taught there, focusing on finite element analysis and 3D modelling, it was a fun time. But soon after taking my first steps in typical mechanical engineering jobs, it was clear to me that I had to go back to coding. So I decided to take the offer to work at my friend’s early stage startup while studying programming on my own, then joined Hack Reactor where I got more full stack experience.

As a software engineer, I have developed a few full-stack web applications using mostly full-stack JavaScript framework including React and Node.js. Some databases I am familiar with are MongoDB, MySQL, and PostgreSQL. I have worked with AWS, but am also currently exploring GCP. I have also done some back-end optimization using Nginx and New Relic which I enjoyed a lot. (Checkout my projects below!) These days, I am working on relearning Python.

In my free time, I enjoy going on a run, playing piano, and watching movies!
